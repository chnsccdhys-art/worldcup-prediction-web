# World Cup Prediction Web

A static World Cup prediction tool for four matches.

Open `index.html` in a browser, or deploy the folder to GitHub Pages.

Current model presets:

- Strength priority: XG/XGA 50%, odds market 20%, lineup 10%, form/statistics 10%, context 10%.
- Odds priority: XG/XGA 20%, odds market 50%, lineup 10%, form/statistics 10%, context 10%.

API keys are not stored in this project.
